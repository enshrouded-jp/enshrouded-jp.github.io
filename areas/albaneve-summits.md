# Albaneve Summits（アルバネーヴ・サミッツ）

> 📌 情報源: [Enshrouded Wiki](https://enshrouded.wiki.gg/wiki/Biomes)

高地の寒冷バイオーム。最も厳しい環境で、強力な敵が多数生息します。

![Albaneve Summits ボス戦](https://shared.akamai.steamstatic.com/store_item_assets/steam/apps/1203620/01ab6c97b4c33591719222c28bb74083e722f064/ss_01ab6c97b4c33591719222c28bb74083e722f064.1920x1080.jpg?t=1776758849)

## 基本情報

| 項目 | 内容 |
|------|------|
| 難易度 | ★★★★★（後半・高難度） |
| 主要敵勢力 | 強力なFell、クモ、雪豹、神話生物 |
| 主要採集素材 | コニファー木材（Conifer Wood）、紫水晶（Amethyst）、花崗岩（Granite）、黒曜石（Obsidian）、銀（Silver） |
| 特徴 | 高地・寒冷環境。高価値素材が豊富だが敵が強力 |

## 採集できる主要素材

- **コニファー木材**: 高品質な建材
- **紫水晶（Amethyst）**: 魔法系クラフトに使用
- **黒曜石（Obsidian）**: 高品質な武器・ツール素材
- **銀（Silver）**: アルケミスト系クラフト・装備強化
