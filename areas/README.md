# マップ・エリア概要

> 📌 情報源: [Enshrouded Wiki - Biomes](https://enshrouded.wiki.gg/wiki/Biomes)

Enshroudedの世界「Embervale（エンバーヴェイル）」は6つのバイオームで構成されています。各バイオームで出現する敵、採集できる素材、出会える生存者NPCがそれぞれ異なります。

## バイオーム一覧

| バイオーム | 難易度 | レベル帯 | 特徴 | 主要敵勢力 | 主要素材 |
|----------|------|--------|------|-----------|---------|
| [Springlands](springlands.md) | ★☆☆☆☆ | 1〜10 | 草原・森林。ゲーム開始地点 | Scavengers, Vukah, Fell, Hollow | 塩、燧石、植物繊維 |
| [Revelwood](revelwood.md) | ★★☆☆☆ | 10〜15 | 密林・Blackmire沼地。廃都Pikemead's Reach | Fell, Scavengers, Vukah | 硬木、銅鉱石、粘土、アンバー |
| [Nomad Highlands](nomad-highlands.md) | ★★★☆☆ | 15〜20 | 石灰岩高原、巨大火山2つ（Pillars of Creation） | Vukah, Scavengers, Fell | 錫、石灰岩、化石骨 |
| [Kindlewastes](kindlewastes.md) | ★★★★☆ | 20〜30 | 砂漠・砂岩台地。シュラウドが南北を分断 | Scavengers, Fell, Vukah | パームウッド、サフラン、鉄鉱石、硫黄 |
| [Albaneve Summits](albaneve-summits.md) | ★★★★★ | 30〜40 | 高山・寒冷地帯。防寒装備必須 | 強力なFell各種、クモ、雪豹、Cyclops | コニファー材、紫水晶、黒曜石、銀 |
| [Veilwater Basin](veilwater-basin.md) | ★★★★★ | 40〜45 | 熱帯ジャングル・淡水湖（Update 8追加） | Drak勢力 | 熱帯木材、コーヒー豆、アクアマリン、金鉱石 |

## 進行の目安

Springlandsでチュートリアルを経て、以下の順に難易度が上がります：

```
Springlands（1-10）
  → Revelwood（10-15）
    → Nomad Highlands（15-20）
      → Kindlewastes 北部（20-25）→ 南部（25-30）
        → Albaneve Summits（30-40）
          → Veilwater Basin（40-45、Update 8）
```

各エリアへ進む前に必要なFlame Levelを確認してください：
- Revelwood: Flame Level 3
- Nomad Highlands: Flame Level 4
- Kindlewastes: Flame Level 5
- Albaneve Summits: Flame Level 6〜8

## 生存者（NPC職人）の初期救出場所

生存者はバイオーム内の Ancient Vault に囚われています。最初に救出できる6名はすべてSpringlandsに配置されています：

| 生存者 | バイオーム |
|--------|----------|
| ブラックスミス（Blacksmith） | Springlands |
| アルケミスト（Alchemist） | Springlands |
| カーペンター（Carpenter） | Springlands |
| コレクター（Collector） | Springlands |
| ファーマー（Farmer） | Springlands |
| ハンター（Hunter） | Springlands |
| バード（Bard） | Revelwood |

序盤はまずSpringlandsのAncient Vault（5か所）を全制圧してから次のバイオームへ進むことを推奨します。

## Ancient Spire（エンシェントスパイア）について

各バイオームにAncient Spireが設置されており、登頂・起動することで周辺のマップ霧が晴れます。新バイオームに入ったらまず最寄りのAncient Spireを目指すのが基本戦略です。

各バイオームの詳細マップ・POI・ダンジョン・素材情報は各ページを参照してください。
