# マップ・エリア概要

> 📌 情報源: [Enshrouded Wiki - Biomes](https://enshrouded.wiki.gg/wiki/Biomes)

Enshroudedの世界「Embervale（エンバーヴェイル）」は6つのバイオームで構成されています。各バイオームで出現する敵、採集できる素材、出会える生存者NPCがそれぞれ異なります。

## バイオーム一覧

| バイオーム | 難易度 | 特徴 | 主要敵勢力 | 主要素材 |
|----------|------|------|-----------|---------|
| [Springlands](springlands.md) | ★☆☆☆☆ | 草原・森林。ゲーム開始地点 | Scavengers, Vukah, Fell | 塩、燧石 |
| [Revelwood](revelwood.md) | ★★☆☆☆ | 密林・Blackmire沼地 | Fell各種 | 硬木、銅、粘土 |
| [Nomad Highlands](nomad-highlands.md) | ★★★☆☆ | 石灰岩地形、巨大火山2つ | Vukah, Scavenger定住地 | 錫、石灰岩、化石骨 |
| [Kindlewastes](kindlewastes.md) | ★★★★☆ | 砂漠、サボテン、砂岩台地 | 野生動物、Scavenger勢力 | パームウッド、サフラン、鉄鉱石 |
| [Albaneve Summits](albaneve-summits.md) | ★★★★★ | 高地・寒冷環境 | 強力なFell、クモ、雪豹、神話生物 | コニファー木材、紫水晶、花崗岩、黒曜石、銀 |
| [Veilwater Basin](veilwater-basin.md) | ★★★★★ | 熱帯ジャングル・淡水湖（Update 8追加） | Drak勢力 | 熱帯木材、コーヒー豆、アクアマリン、金鉱石 |

## 進行の目安

Springlandsでチュートリアルを経て、Revelwood → Nomad Highlands → Kindlewastes → Albaneve Summits の順に難易度が上がります。Veilwater Basin（Update 8）はLv45上限に対応する最高難度エリアです。

## 生存者（NPC職人）の初期救出場所

生存者はバイオーム内の Ancient Vault に囚われています。特に Springlands に最初の生存者（Blacksmith、Alchemistを含む複数名）が配置されており、序盤はまず Springlands の Ancient Vault 探索が優先です。

各バイオームの詳細マップ・POI・ダンジョン情報は各ページを参照してください。
