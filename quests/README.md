# クエスト概要

> 📌 情報源: [enshrouded.wiki.gg/wiki/Quests](https://enshrouded.wiki.gg/wiki/Quests)

Enshroudedのクエストは大きく以下に分類される。総クエスト数は **257** に及ぶ。

## クエストの種類

| 種類 | 説明 |
|------|------|
| **メインクエスト** | The Flame および各職人（Craftspeople）が提供。ストーリー進行・新バイオーム解放・重要NPCのアンロックが報酬 |
| **サイドクエスト** | 村人（Villagers）やLore発見によって解放される補助的クエスト |
| **Ancient Vault** | 古代遺跡ダンジョン型の探索クエスト。生存者救出と強力な装備が目的 |

## ページ一覧

- [メインクエスト](main.md): ストーリー進行の軸
- [サイドクエスト](side.md): 生存者救出・地域クエスト等
- [Ancient Vaults](ancient-vaults.md): 古代遺跡ダンジョン型クエスト

## 進行の目安

1. The Flameのクエストに従い拠点を構築
2. Ancient Vaultで各職人（生存者）を救出
3. 各職人のクエストを並行して進める
4. Lore（古代の記録）を集め、サイドクエストを解放
5. 全バイオームを解放し、Flame Levelを最大化
