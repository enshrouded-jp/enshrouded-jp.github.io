# Ancient Vault（エンシェント・ヴォールト）

> 📌 情報源: enshrouded.wiki.gg/wiki/Ancient_Vault, enshrouded.wiki.gg/wiki/Elixir_Well, enshrouded.wiki.gg/wiki/Shroud

## 概要

**エリクサー・ウェル（Elixir Well）**はエンバーヴェール各地に点在するシュラウドの発生源となる井戸型ダンジョン。ゲーム内では計19箇所が存在する。Shroud（霧）はこのエリクサー・ウェルの深部から湧き出しており、底に宿るシュラウド・ルート（Shroud Root）を破壊することがクリア条件となる。

> 公式wikiの「Ancient Vault」ページはスタブ状態（2026年1月時点）のため、Elixir Wellおよびボス情報から補完しています。

## 攻略の流れ

1. **シュラウドに侵入する**  
   エリクサー・ウェルはシュラウドに包まれた地点に位置する。侵入するとタイマーカウントが開始される。タイマーは画面上部に表示され、0になるとシュラウドに飲み込まれる（装備ロストはなく拠点へ強制帰還）。

2. **ボスを撃破する**  
   ウェルの内部にはボスが配置されており、到達前に複数の雑魚敵も出現する。各エリアのボスを倒すことがシュラウド・ルート破壊への道筋となる。

3. **シュラウド・ルートを破壊する**  
   ボス撃破後、ウェルの深部にある Shroud Root を斧などで破壊する。破壊するとエリア内の霧が晴れ、周辺の全フレイムボーンにスキルポイントが付与される。

## 出現するボス

各エリクサー・ウェルにはバイオームに対応したボスが出現する。以下は確認されている対応関係。

| バイオーム | 主な出現ボス |
|---|---|
| Springlands | フェル・サンダーブルート（Fell Thunderbrute） |
| Low Meadows | フェル・サンダーブルート（Fell Thunderbrute） |
| Revelwood | フェル・ウィスプワイバーン（Fell Wispwyvern）、フェル・クリッタークイーン（Fell Critter Queen） |
| Nomad Highlands | フェル・モンストロシティ（Fell Monstrosity） |
| Kindlewastes | フェル・モンストロシティ（Fell Monstrosity）、フェル・シックルサイス（Fell Sicklescythe） |
| Albaneve Summits | フェル・サイクロプス（Fell Cyclops）、フェル・モンストロシティ（Fell Monstrosity） |
| Veilwater Basin | フェル・モンストロシティ（Fell Monstrosity） |

## タイマー管理の注意点

- **レスポーンポイント（Respawn Point）** の近くにいる間は無限にタイマーが維持される
- **キャニスター（Canister）** や**ポーション（Potion）** を使用することで任意にタイマーを回復できる
- **デッドリー・シュラウド（Deadly Shroud）**（赤い霧）はタイマーの減少が通常より大幅に速い。立ち入らないよう注意

## 報酬

- **スキルポイント**: 破壊時に周辺の全フレイムボーンへ付与
- **ボスドロップ**: 各ボス固有のヘッド素材（フレイム・アルターのアップグレードに必要）、レジェンダリー・ルーン、シュラウド・コア等
- **エリア解放**: シュラウドが晴れてタイマーなしで行動可能になる
