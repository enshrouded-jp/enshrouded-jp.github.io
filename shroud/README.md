# Shroud（霧）概要

> 📌 情報源: enshrouded.wiki.gg/wiki/Shroud

シュラウド（Shroud）はエンバーヴェール（Embervale）を覆う有害な霧。プレイヤーが侵入すると**制限時間付き**の探索が始まり、タイマーが切れると死亡して拠点へ強制帰還する（装備ロストはなし）。

## Shroudの基本ルール

- **滞在時間制限**: 侵入と同時にタイマーが減少し始める
- **デッドリー・シュラウド**: 赤い霧はタイマーの減少が大幅に速い。立ち入り注意
- **タイマー回復**: レスポーンポイント付近での滞在・キャニスター・ポーションで回復可能
- **タイマー切れ**: 死亡 → 拠点リスポーン（装備ロストなし）
- **恒久解放**: エリクサー・ウェルのシュラウド・ルートを破壊すると、そのエリアの霧が永続的に晴れる

## Shroud内のコンテンツ

- **エリクサー・ウェル（Elixir Well）**: シュラウドの発生源。底の Shroud Root を破壊するとスキルポイント獲得とエリア解放
- **フェル系敵（Fell Enemies）**: シュラウドに感染した強敵が多数出現
- **固有素材**: シュラウド内でのみ入手できる素材が存在する
