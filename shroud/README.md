# Shroud（霧）概要

> 📌 本ページはサイト初期構築時点のスタブです。

Shroud（霧）はEnshroudedの世界を覆う有害な霧。プレイヤーが侵入すると**制限時間付き**の探索が始まり、タイマー切れで死亡する（装備ロストはなく拠点へ強制送還）。

## Shroudの基本ルール

- **滞在時間制限**: 時間経過でタイマーが減少
- **タイマー延長**: 特定のアイテムや仕組みで延長可能（詳細は情報収集中）
- **タイマー切れ時**: 死亡 → 拠点リスポーン

## Shroud内のコンテンツ

- Shroud Root: 破壊で霧が晴れる発生源
- Shroud Core: より強力な拠点的存在（情報収集中）
- 固有の敵・素材
