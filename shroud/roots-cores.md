# Shroud Roots / Shroud Cores（シュラウド・ルート / シュラウド・コア）

> 📌 情報源: enshrouded.wiki.gg/wiki/Shroud_Root, enshrouded.wiki.gg/wiki/Elixir_Well, enshrouded.wiki.gg/wiki/Shroud, 各ボスwikiページ

## Shroud Root（シュラウド・ルート）

**シュラウド・ルート（Shroud Root）** はシュラウド（霧）の発生源となる構造物。各地の**エリクサー・ウェル（Elixir Well）** の深部に宿っており、計19箇所のウェルに存在する。破壊することでそのエリアの霧が晴れ、以降はタイマーなしで行動可能になる。

### 破壊の流れ

1. **エリクサー・ウェルに到達する**  
   シュラウドに侵入してウェルの場所まで進む。タイマーが減少し始めるため、タイマー管理（後述）を行いながら進む。

2. **ボスを撃破する**  
   ウェルの内部にはバイオームに対応したボスが配置されており、ボスを倒さないとシュラウド・ルートにアクセスできない場合が多い。

3. **シュラウド・ルートを破壊する**  
   ウェルの最深部にある Shroud Root を攻撃して破壊する。斧（Axe）系の武器が有効。複数の雑魚敵も周囲に出現するため注意。

### 破壊報酬

- **スキルポイント（Skill Points）**: 破壊時に付近にいる全フレイムボーンへ付与される
- **エリア解放**: 周辺のシュラウドが永続的に消滅し、霧のないエリアとして使えるようになる
- **ボスドロップ（事前撃破）**: ヘッド素材、シュラウド・コア、レジェンダリー・ルーン等

## Shroud Core（シュラウド・コア）

**シュラウド・コア（Shroud Core）** はボスが確率でドロップするクラフト素材。シュラウド・ルートそのものとは別のアイテムであり、エリア解放とは直接関係しない。

主にフレイム・アルター（Flame Altar）のアップグレードや各種クラフトに使用される。

| アイテム | 概要 |
|---|---|
| Shroud Root | ウェル深部の構造物。破壊するとエリアが永続解放 |
| Shroud Core | ボスドロップ素材。クラフトに使用 |
| Shroud Nexus | 上位クラフト素材。一部ボスからドロップ |

## Shroud内でのサバイバル注意点

シュラウド内での行動にはタイマーによる制限がある。

### タイマーの基本

- シュラウドエリアに入ると画面上部にカウントダウンタイマーが表示される
- タイマーが0になると**シュラウドに飲み込まれ死亡**する（装備ロストなし、拠点へ強制帰還）
- **デッドリー・シュラウド（Deadly Shroud）**（赤い霧）はタイマーの減少が通常の霧より大幅に速い

### タイマーを延長・回復する方法

| 方法 | 効果 |
|---|---|
| レスポーンポイント（Respawn Point）の近く | タイマーが維持される（無限） |
| キャニスター（Canister）使用 | タイマーを一定量回復 |
| ポーション（Potion）使用 | タイマーを一定量回復 |
| フレイム・アルターのアップグレード | シュラウド耐性が恒久的に向上 |

### その他の注意点

- シュラウド内には通常の敵に加えて**フェル系の敵**が多数出現する
- **爆発するパフボール（Explosive Puffballs）** が生えている地点があり、接触するとダメージを受ける
- ウェル内部には毒スライム（Toxic Slime）やトラップが設置されている場合がある
- ガード・オブ・ザ・ノース（Guard of the North）防具セットのように、装備によっては最大シュラウド時間が**減少**するものがあるため注意
