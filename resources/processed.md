# 加工素材

> 📌 情報源: [enshrouded.wiki.gg - Crafting](https://enshrouded.wiki.gg/wiki/Crafting) / [Wood Planks](https://enshrouded.wiki.gg/wiki/Wood_Planks) / [Fabric](https://enshrouded.wiki.gg/wiki/Fabric) / [Charcoal](https://enshrouded.wiki.gg/wiki/Charcoal) / [Leather](https://enshrouded.wiki.gg/wiki/Leather) / [Resin](https://enshrouded.wiki.gg/wiki/Resin)

## 木材加工品

### ウッドプランク（Wood Planks / 木材板）

- **設備**: Table Saw（テーブルソー）
- **レシピ**: 任意の Wood Logs×1 → Wood Planks×2（1分）
- **用途**: 建築・家具・各種設備の製作に必須。大工（Carpenter）・農民（Farmer）のほぼ全レシピで使用

---

## 繊維加工品

### リネン（Linen / 亜麻布）

- **素材**: Flax（亜麻）から生産
- **設備別レシピ**:

| 設備 | 必要素材 | 時間 | 産出量 |
|------|----------|------|--------|
| Hand Spindle | Flax×2 | 10秒 | Linen×1 |
| Spinning Wheel | Flax×1 | 30秒 | Linen×1 |
| Spinning Machine | Flax×10 | 4分 | 複数 |

### ファブリック（Fabric / 布）

- **設備**: Loom（織機）
- **レシピ**: Linen×2 → Fabric×1（2分）
- **入手バイオーム**: キンドルウェイスツ（Kindlewastes）以降
- **用途**: 防具・グライダー・ベッド・カーペット・帆など

---

## 皮革加工品

### ドライドファー（Dried Fur / 乾燥毛皮）

- **設備**: Drying Rack（乾燥ラック）
- **レシピ**: Animal Fur（動物の毛皮）を乾燥
- **用途**: Leather（皮革）製造の主材料

### レザー（Leather / 皮革）

- **設備**: Tanning Station（タンニングステーション）
- **レシピ**: Dried Fur×10 + Salt（塩）×20 + Ammonia Gland（アンモニア腺）×2 → Leather×10（15分）
- **入手バイオーム**: ノマドハイランズ（Nomad Highlands）以降
- **用途**: 防具（アサシン系・ガーディアン系等）・家具・武器・グライダー

### ハードンドレザー（Hardened Leather / 硬化皮革）

- **用途**: 上位防具・装備の材料
- **製法**: 情報収集中

## 燃料・化学系加工品

### チャーコール（Charcoal / 木炭）

- **設備**: Charcoal Kiln（木炭窯）
- **レシピ**: Any Wood×17 + Any Dirt×3 → Charcoal×15（5分）
- **アンロック条件**: Stone（石）入手後
- **用途**: Smelterでの金属製錬の燃料。Bronze Bars等の製作に必須

### リジン（Resin / 樹脂）

- **入手方法**: 木の伐採時に副産物として入手。特に赤い葉の木（Revelwood Red）から多く取れる
- **栽培**: Revelwood Red Leaf Seedlingを育てると効率よく収集可能
- **用途**: 建築（松明・暖炉）・魔法アイテム（ファイアボール・フレア等）・矢・装甲品

### ウッドアシッド（Wood Acid）

- **入手方法**: 錬金術師（Alchemist）でアンロックされるレシピで生産
- **用途**: Tin Bars（スズインゴット）の製錬に必須の補助材料

### エンシュラウデッドオイル（Enshrouded Oil）

- **設備**: Laboratory（ラボラトリー）
- **レシピ**: Shroud Particles×5 + Coal Powder×5 + Brittle Shroud Flakes×5 + Sulfur（硫黄）×1 → Enshrouded Oil×5（2分）
- **アンロック条件**: 錬金術師（Alchemist）でLaboratoryをアンロック後
- **用途**: Ghost Glider・Huge Magic Chest・Lightning Channel等の高級クラフト

---

## 金属加工品

### メタルシーツ（Metal Sheets / 金属板）

- **設備**: Forge（鍛造炉）
- **レシピ**: Charcoal×2 + Scrap Metal（金属くず）×2 → Metal Sheets×1（2分）
- **用途**: 建築（大型ワゴン・金属ブロック）・道具（ケトル・グラップリングフック部品）
