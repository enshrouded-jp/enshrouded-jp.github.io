# 素材概要

> 📌 情報源: [enshrouded.wiki.gg - Category:Crafting_Materials](https://enshrouded.wiki.gg/wiki/Category:Crafting_Materials)

Enshroudedの素材は**採集素材・鉱石/インゴット・加工素材**の3カテゴリに大別されます。

- [採集素材](gathering.md): 木材・植物・繊維・動物・Shroud素材など
- [鉱石・インゴット](ores-ingots.md): 採鉱で得られる鉱石と精錬インゴット
- [加工素材](processed.md): 中間クラフト素材（板材・布・木炭など）

wikiでは275種類以上のクラフト素材が確認されています。素材ごとの入手バイオーム・用途は各ページ参照。

## カテゴリ別概要

| カテゴリ | 代表素材 | 主な入手方法 |
|----------|----------|--------------|
| 採集素材 | Wood Logs、Flax、Resin | 伐採・採集・狩猟 |
| 鉱石 | Copper/Iron/Tin/Silver/Gold | 鉱山・洞窟での採掘 |
| インゴット | Copper/Iron/Bronze/Steel/Silver/Gold Bars | Smelter・Blast Furnaceで製錬 |
| 加工素材 | Wood Planks、Fabric、Charcoal、Leather | 各種設備でクラフト |
