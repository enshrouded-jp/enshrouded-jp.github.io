## 戦闘用語

| 英語 | カタカナ / 日本語 | 備考 |
|------|-----------------|------|
| Focus Bar | フォーカスバー | 通常攻撃で蓄積するゲージ。満タンで武器固有のSpecial Abilityを発動 |
| Special Ability | スペシャル・アビリティ | フォーカスバー満タン時に発動できる武器固有の特殊技 |
| Heavy Attack | ヘビー・アタック | Update 8で全近接武器に追加。ダメージ2倍。スタンバーへの効果も大 |
| Stun Bar | スタン・バー | 敵のスタン耐性を示すゲージ。ヘビー・アタックで効率よく削れる |
| Strategic Maneuver | ストラテジック・マヌーバー | ロックオン中に敵の背後へ回避できる新スキル（Update 8追加） |
| Double Jump | ダブルジャンプ | スキルツリーで習得可能。Update 8でツリー中央寄りに移動し取得しやすくなった |
| Lock-on | ロックオン | 特定の敵に照準を固定する戦闘機能 |
| Updraft | アップドラフト | グライダー使用中に上昇できる気流。Update 8で最高高度が約15%上昇 |

## クラフト・建築

| 英語 | カタカナ / 日本語 | 備考 |
|------|-----------------|------|
| Workbench | ワークベンチ | 最初のクラフトテーブル。使用時に全装備が即時修復される。素材: String×3 + Wood Log×8 |
| Magic Chest | マジック・チェスト | 建築時にチェスト内の素材を直接選択して配置できるチェスト（Update 8追加） |
| Quick Stack Station | クイック・スタック・ステーション | 近隣チェストへ素材を自動搬出する施設（Update 8追加） |
| Manual Crafting | マニュアル・クラフティング | インベントリ画面から道具を使わず手作業でクラフトする方法 |
| Rune | ルーン | Update 8で追加された装備強化素材。防具・盾・ワード・リング・ツールに使用可能 |
| Shroud Wood | シュラウドウッド | Shroudエリアで採取できる木材。グライダーのクラフトに必要 |
| Shroud Spore | シュラウドスポア | Shroudエリアで採取できる胞子。グライダーのクラフトに必要 |
| Animal Husbandry | アニマル・ハズバンドリー | 動物の飼育システム。Farmerが管理 |
| Water Wheel | ウォーター・ウィール | Update 7で追加された水力建築設備 |
| Glider | グライダー | 空を飛ぶための装備。ワークベンチでクラフト可能。素材: Shroud Wood×8 + Animal Pelt×2 + String×2 + Shroud Spore×2 |
