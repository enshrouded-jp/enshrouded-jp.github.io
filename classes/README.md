# クラス・スキルツリー概要

> 📌 情報源: [Enshrouded Wiki - Skills](https://enshrouded.wiki.gg/wiki/Skills)
> ⚠️ Update 8にてスキルツリーが大幅リワーク済み。既存セーブのスキルポイントは全返還されています。

## スキルツリーの基本

Enshroudedには固定のクラス制は存在しません。3本のスキルツリーからポイントを自由に振ることでビルドを構築します。ハイブリッドも可能です。

## スキルポイントの獲得方法

| 方法 | 獲得量 |
|------|------|
| レベルアップ | 2ポイント / レベル |
| Shroud Root 撃破 | 1ポイント |
| Elixir Well 完了 | 3ポイント |

## サブクラス詳細

### Beastmaster（ビーストマスター）― Ranger Tree

**プレイスタイル**: 野生動物を制御・味方化しながら戦う生存・支援型。毒耐性を高めて自身の生存力を上げつつ、周囲の動物を盾・攻撃役として活用する。ソロ探索やオープンワールドでの生存に向いている。

| スキル名 | コスト | 効果 |
|---------|--------|------|
| Snake Eater（スネーク・イーター） | 2pt | 毒耐性+5%（毒ダメージ軽減） |
| Mithridatist（ミスリダティスト） | 3pt | 毒耐性+10%、25%の確率で毒を無効化 |
| Calm Spirit（カーム・スピリット） | 3pt | 50m以内の野生動物を無害化（攻撃しない限り） |
| Vukah Language（ヴカ語） | 3pt | 50m以内のVukahが攻撃してこなくなる |
| Beast Master（ビーストマスター） | 4pt | 攻撃を受けると50m以内の野生動物が敵を援護攻撃する |
| Vukah Culture（ヴカ文化） | 4pt | Beast Masterと同様の効果がVukah種族にも適用 |
| Endurance（エンデュランス） | 1pt | スタミナ+10 |
| Dexterity（デクスタリティ） | 1pt | 弓・短剣ダメージ+5% |
| Endurance Of The Flame | 5pt | フレームレベル2上昇ごとにエンデュランス+1 |

---

### Trickster（トリックスター）― Mage Tree

**プレイスタイル**: 魔法攻撃とダメージ反射を組み合わせた攻撃型。受けたダメージを火炎として反射し、魔法クリティカルでスタンを発生させる。グライダー中の上昇スキルで高い機動性も持つ。魔法ビルドの中では攻撃的・立ち回り重視の構成。

| スキル名 | コスト | 効果 |
|---------|--------|------|
| Counterstrike（カウンターストライク） | 2pt | ダメージを受けた際、20%の確率で受けたダメージの50%を火炎として攻撃者に反射 |
| Begone（ビゴーン） | 3pt | 魔法パンチで敵を吹き飛ばし・スタンさせる（消費マナ30） |
| Terror（テラー） | 4pt | 魔法クリティカルヒット時に対象を4秒間スタン |
| Shock Resistance（ショック耐性） | 2pt | ショック耐性+5%（複数取得可） |
| Updraft（アップドラフト） | 4pt | グライダー滑空中にジャンプすると上昇する（消費マナ100） |
| Spirit（スピリット） | 1pt | 最大マナ+20 |
| Intelligence（インテリジェンス） | 1pt | 魔法ダメージ+5% |
| Arcane Concentration | 5pt | フレームレベル2上昇ごとにスピリット+1 |

## 主要スキル（例）

- **Sneak Attack（スニークアタック）**: 背後からの攻撃でダメージ+900%
- **Merciless Attack（マーシレスアタック）**: ブロック中の敵を攻撃してスタンバーを溜める
- **Strategic Maneuver（ストラテジック・マニューバー）**: ロックオン時に敵の背後へ回避（Update 8追加）
- **Double Jump**: ツリー中央に移動（Update 8リワーク後、アクセスしやすい位置）
