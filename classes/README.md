# クラス概要

> 📌 Enshroudedには固定のクラス制は存在せず、スキルツリーを自由に振ることでビルドを構築します。本サイトでは慣習的な**ビルド指向6種**に分けて解説します。

## スキルツリーとクラスの関係

- レベルアップでスキルポイントを獲得
- スキルツリーは**どの方向からも振れる**
- 複数方向に振るハイブリッドビルドも可能

## ビルド指向6種

| 指向 | 主要武器 | 役割 |
|------|---------|------|
| [Warrior](warrior.md) | 近接（両手剣・片手剣＋盾） | タンク・前衛 |
| [Battlemage](battlemage.md) | 近接＋魔法 | ハイブリッド |
| [Ranger](ranger.md) | 弓 | 遠距離DPS |
| [Assassin](assassin.md) | 片手剣・短剣 | 高速前衛DPS |
| [Wizard](wizard.md) | 杖・魔法 | 遠距離魔法DPS |
| [Healer](healer.md) | 支援魔法中心 | サポート（Co-op向け） |

具体的なスキル振りは [ビルド例](builds.md) を参照。
