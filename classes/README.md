# クラス・スキルツリー概要

> 📌 情報源: [Enshrouded Wiki - Skills](https://enshrouded.wiki.gg/wiki/Skills)
> ⚠️ Update 8にてスキルツリーが大幅リワーク済み。既存セーブのスキルポイントは全返還されています。

## スキルツリーの基本

Enshroudedには固定のクラス制は存在しません。3本のスキルツリーからポイントを自由に振ることでビルドを構築します。ハイブリッドも可能です。

## スキルポイントの獲得方法

| 方法 | 獲得量 |
|------|------|
| レベルアップ | 2ポイント / レベル |
| Shroud Root 撃破 | 1ポイント |
| Elixir Well 完了 | 3ポイント |

## 3本のスキルツリー

### Warrior Tree（近接系）

| サブクラス | 特徴 |
|-----------|------|
| **Warrior（ウォリアー）** | バランスの良い近接DPS |
| **Tank（タンク）** | 高耐久・盾受けスペシャリスト |
| **Barbarian（バーバリアン）** | 両手武器による高火力 |
| **Athlete（アスリート）** | スタミナ・機動力特化 |

### Ranger Tree（遠距離・生存系）

| サブクラス | 特徴 |
|-----------|------|
| **Ranger（レンジャー）** | 弓遠距離DPS |
| **Assassin（アサシン）** | ステルス・クリティカル特化 |
| **Beastmaster（ビーストマスター）** | 情報収集中 |
| **Survivor（サバイバー）** | 生存・クラフト特化 |
| **Athlete（アスリート）** | スタミナ・機動力特化 |

### Mage Tree（魔法系）

| サブクラス | 特徴 |
|-----------|------|
| **Wizard（ウィザード）** | 魔法遠距離DPS |
| **Healer（ヒーラー）** | 回復・支援（Co-op向け） |
| **Battlemage（バトルメイジ）** | 近接＋魔法ハイブリッド |
| **Trickster（トリックスター）** | 情報収集中 |

## 主要スキル（例）

- **Sneak Attack（スニークアタック）**: 背後からの攻撃でダメージ+900%
- **Merciless Attack（マーシレスアタック）**: ブロック中の敵を攻撃してスタンバーを溜める
- **Strategic Maneuver（ストラテジック・マニューバー）**: ロックオン時に敵の背後へ回避（Update 8追加）
- **Double Jump**: ツリー中央に移動（Update 8リワーク後、アクセスしやすい位置）
