## 弓レンジャービルド（Archer）

**難易度**：初〜中級者向け　**評価**：現環境最強クラス

Rangerビルドはダメージ・機動力・生存性のバランスが高水準で、ソロ・Co-op問わず活躍できます。Hack the Minotaurの評価では全ビルド中最もおすすめとされています。

### スキルツリー振り分け
**Rangerツリー（主）**：Marksman、Bow Specialization、Multishot、Chain Reaction、Ricochets  
**Assassinツリー（副）**：Shell Shock、Expose Weakness、Fatal Precision  
**Healerツリー（補）**：Water Aura、Waters of Life、Blink  
**センター**：Double Jump、Backstab

### 推奨装備
| 部位 | 装備 |
|-----|------|
| 頭 | Quetzal Hunter Helmet（遠距離クリティカル+18%） |
| 胸 | Sunpiercer Chestplate（HP+360） |
| 手 | Skull Stalker Gloves（弓ダメージ+9%） |
| 脚 | Quetzal Hunter Pants（スプリント+12%） |
| 足 | Quetzal Hunter Shoes（スタミナ回復+18） |
| 武器 | Ignited Bow ＋ Staff（回復用） |
| リング | Gemini Ring ×2（HPリーチ+7%） |
| 宝石 | Piercing または Acrobatics |

### 基本ローテーション
1. 高所または安全な距離を確保
2. Shroud Arrowで開幕DoT（継続ダメージ）付与
3. Chain ReactionやExplosive Arrowでメインダメージ
4. 接近されたらBlinkで離脱→Double Jumpで高所へ
5. Water Auraのパッシブ回復でHPを維持

## 近接タンクビルド（Barbarian Tank）

**難易度**：初〜中級者向け　**評価**：ソロ安定・Co-op前衛向け

両手武器（グレートソードまたはアックス）とWarrior系スキルを組み合わせた、攻防一体の近接ビルドです。Battle HealとBlood Rageによる自動回復が安定感の核心です。

### スキルツリー振り分け
**Barbarianツリー（主）**：Barbarian、Heavy Hitter、Whirlwind Crescendo、Two-Handed Specialization  
**Warriorツリー（副）**：Warrior、Battle Heal（クリティカル時HP+5%）、Veteran  
**Athleteツリー（補）**：Crash Down Attack、Uppercut  
**Tankツリー（補）**：Tough Skin、Earth Aura（近接タンク型の場合）  
**センター**：Double Jump、Merciless Attack、Backstab

### 主要スキル解説
- **Blood Rage**：低HP敵を倒した際にメレーダメージ+20%が20秒間持続。積極的に敵を仕留めることでバフを維持
- **Battle Heal**：クリティカルヒット時に最大HP5%回復。継続的なクリティカルで事実上の自動回復
- **Merciless Attack**：スタン（オーバーパワー）状態の敵に500%ダメージ。ヘビーアタックでスタンを狙ってから発動

### 推奨装備
| 部位 | 装備 |
|-----|------|
| 頭 | Sunpiercer Helmet（メレークリティカル+18%、クリティカルダメージ+15%） |
| 胸 | Sunpiercer Chestplate（HP+360） |
| 手 | Sunpiercer Gloves（両手武器ダメージ+18%） |
| 脚 | Golden Bulwark Pants（HP+135、スタミナ回復） |
| 足 | Quetzal Hunter Shoes（スタミナ回復） |
| 武器 | Champion's Axe（切撃+20%、HPリーチ+5%、クリティカル+5%） |
| リング | Gemini Ring ×2（HPリーチ+7%） |
| 宝石 | Gem of Bleeding（出血DoT） |

### 基本ローテーション
1. スプリント接近→ジャンプアタックでAoE発動
2. 通常コンボ＋ヘビーアタック（長押し）でスタンゲージを削る
3. スタン発生したら**Merciless Attack**で大ダメージ
4. **Blood Rage**のバフ維持を意識して低HPの敵を優先的に仕留める
5. HP回復はBattle Heal（クリティカル）＋Gemini Ring（HPリーチ）で自動維持

## 魔法アタッカービルド（Wizard DPS）

**難易度**：中〜上級者向け　**評価**：範囲殲滅最強クラス

杖（Staff）を主力としたAoE魔法特化ビルドです。Intelligence最大化で呪文威力と水オーラ回復を同時に強化します。Blinkの使いこなしが難易度を左右します。

### スキルツリー振り分け
**Wizardツリー（主）**：Wizard、Chain Hit、Mass Destruction、Radiant Aura、Arsonist  
**Healerツリー（副）**：Water Aura、Waters of Life、Life Burst  
**Battlemageツリー（補）**：Blink、Emergency Blink、Blink Attack  
**Tricksterツリー（補）**：Begone!、マナ回復スキル

### 推奨装備
| 部位 | 装備 |
|-----|------|
| 頭 | Circlet of Lightning（ショックダメージ+15%、クリティカル+21%） |
| 胸〜足 | Tidecaller Set（魔法特化アーマー） |
| 武器 | Violet Storm（杖）＋ Mythic Wand（サブ） |
| リング1 | Ring of the Ancients（全ステータス+1、魔法ダメージ+5%） |
| リング2 | Gemini Ring（HPリーチ+7%） |
| 宝石 | Gem of Burning（火ダメージ強化） |

### 基本ローテーション
1. Radiant Auraを発動（継続火ダメージ設置）
2. Chain HitでAoEダメージを連鎖展開
3. Mass Destructionで範囲を拡大しながら追加呪文
4. 敵が接近したらBlinkで後退→Blink Attack発動
5. マナ不足ならワンドに切り替えて回収、Water Auraのパッシブ回復でHP維持

## ステルスアサシンビルド（Poison Assassin）

**難易度**：上級者向け　**評価**：単体バーストに特化、操作技術が問われる

短剣と毒を組み合わせたバースト特化ビルドです。敵の背後から高ダメージのコンボを叩き込み、毒のDoTで削り続けます。被弾に弱い代わりに、一瞬で敵を処理できる爆発力があります。

### スキルツリー振り分け
**Assassinツリー（主）**：Cutthroat（スニーク+100%）、Shell Shock、Expose Weakness、Fatal Precision、Dagger Master  
**Rangerツリー（副）**：Graceful Stride、Slice and Dice、Marksman  
**センター**：Double Jump、Backstab（必須）、Merciless Attack

### 推奨装備
| 部位 | 装備 |
|-----|------|
| 全身 | Bloodfeather Set（クリティカル・回避特化） |
| 武器 | 短剣（Dagger）＋ 弓（牽制・毒矢） |
| リング | Gemini Ring ×2（HPリーチ+7%） |
| 宝石 | Gem of Toxic Betrayal（毒ダメージ付与） |

### 基本コンボ
1. 側面・背後に回り込み**Backstab**発動（位置ボーナス）
2. **Shell Shock**で対象の防御を低下
3. **Expose Weakness**で弱点を露出させる
4. **Fatal Precision**を活かしたクリティカル連打
5. 毒DoTを付与しながら距離を取り→弓でトドメまたは再接近

## Co-op支援ビルド（Support Healer）

**難易度**：初〜中級者向け　**評価**：Co-op専用として評価最高

Co-opプレイで真価を発揮するHealerビルドです。Water Auraのパッシブ回復とHealing Reviveによる蘇生が柱。攻撃よりも味方の生存維持を最優先します。

### スキルツリー振り分け
**Healerツリー（主）**：Intelligence、Healer、Healer II、Water Aura、Waters of Life、Healing Revive、Divine Surge  
**Battlemageツリー（副）**：Blink、Emergency Blink、Unity、Wand Master（マナ効率）  
**Wizardツリー（補）**：Chain Hit、Mass Destruction（攻撃能力の確保）

### 推奨装備
| 部位 | 装備 |
|-----|------|
| 全身 | Waveborne Ritualist Set（回復量+144%） |
| 武器 | Oracle's Everblue（杖）＋ Mythic Wand |
| リング | Ring of the Ancients ×2（全ステータス+1、魔法ダメージ+5%） |

### Co-opでの役割
- **Water Aura**の有効範囲（半径15m）内に味方を収める位置取りが最重要
- 倒れた味方を**即時蘇生**することで戦況の崩壊を防ぐ
- マナ切れ時はワンドに持ち替えてUnity・Wand Masterでマナ回収
- **Martyr**取得後は、自分が倒れた際も味方に最後の回復を提供できる
