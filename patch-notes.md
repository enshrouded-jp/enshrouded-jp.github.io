## Update 8: Forging the Path（2026-04-21）

**v.0.9.1.0 / 1.0前の最後の大型アップデート。**

### 新コンテンツ

- **新バイオーム Veilwater Basin**: 熱帯ジャングル・古代遺跡・淡水湖。新勢力「Drak」が登場
- **レベル上限 Lv45** へ引き上げ
- **Adventure Sharing**: ワールドをサーバーにアップロード/ダウンロードしてコミュニティと共有可能
- **Quick Stack Station**: 近隣チェストへ素材を自動搬出する新施設

### 戦闘リワーク

- **Heavy Attack**: 全近接武器にHeavy Attackを追加（敵とスタンバーへのダメージ2倍）
- **Focus（フォーカス）バー**: 通常攻撃を繰り返すことで蓄積。最大まで溜まると**武器固有のSpecial Ability**を発動可能
- **敵AI改良**: 敵が「調査」段階を経るようになり、実際に視認して初めて戦闘開始。ステルスの戦略性が向上
- **2H斧・ハンマー・グレートソードの攻撃チェーンを全面改良**

### 装備・育成

- **装備アップグレード（ルーン）**: 防具・盾・ワード・指輪・ピッケル・木こり斧がルーンでアップグレード可能に
- **スキルツリーリワーク**: 大幅な変更のため既存キャラのスキルポイントが全返還（再振り分けが必要）。80以上のスキルがアップグレード可能に
- **Double Jump**: ツリー中央に移動し、よりアクセスしやすい位置に
- **Strategic Maneuver（新スキル）**: ロックオン状態で敵の背後に回避できる

### QoL・システム

- **カメラ自動回転**: 新しいカメラ設定。有効時にプレイヤーの移動方向にカメラが自動追従
- **Magic Chest**: 建築時にチェスト内素材を直接選択して配置可能
- **Updraft強化**: グライダーの最高高度が約15%上昇、拠点内では即座のリフト対応
- **Springlands改良**: 序盤エリアのグラフィックと地形多様性が大幅向上

### ホットフィックス

- **Hotfix #37（2026-04-21）**: エラーメッセージの表示問題を修正
- **Hotfix #38（2026-04-22）**: クラッシュ情報収集の改善。ランダム発生するクラッシュ複数を修正

## Update 7: Wake of the Water（2025-11-10）

**v.0.9.0.0 / revision 844087**

### 新コンテンツ

- **新バイオーム Veilwater Basin**: Blackmire〜Albaneve Summitsの間に位置する終盤エリア。熱帯ジャングル・大型湖・川・滝・水中遺跡が広がる。新勢力「Drak」が登場
- **レベル上限 Lv45** へ引き上げ（武器はLv50スケール）
- **水中システム**: アンロック不要で水泳・ダイビングが可能。酸素管理あり（装備・食料で延長可能）。水中では戦闘不可（敵はプレイヤーを攻撃可能）
- **新呪文 Fire Bolt**: 序盤向け単体ターゲット・低コスト魔法

### 新システム

- **釣り（Fishing）**: ロッド・餌・釣りポイントで料理素材・クラフト素材を収集するミニゲーム。釣り師NPCで装備・餌をアップグレード可能
- **グレートソード（Greatsword）**: 両手持ち大剣。序盤エリアから入手可能
- **水建築**: 水バリア・水車（Water Wheel）・水ゲート・ポータブル水ポンプなど水を操作する新建築要素。Flame Altarに「拠点の水を一括除去」ボタン追加
- **Visitor ロール**: マルチプレイヤーで建築物への干渉を防ぐ新権限ロール

### 新スキル（13種）

| スキル名 | 概要 |
|---------|------|
| Upwards Slash Attack | 上方向の斬り上げ攻撃 |
| Splash Dash | 水・湿潤状態に連動したダッシュ攻撃 |
| Whirlwind Crescendo | グレートソード専用AoE旋風斬り |
| Kick | 近接キック攻撃 |
| Fisherman's Resolve | 釣り・水中探索に関連したバフ |
| Wet Dog / Wetter Dog | 濡れ状態を活用する強化スキル |
| Titan Edge | 大型武器のダメージ強化 |
| Last Meal | 瀕死時の緊急回復スキル |
| Explode Weakness | 敵の弱点属性を爆発させる |
| 魔法ダメージ耐性（属性別） | 各属性ダメージを軽減 |
| Shared Misery | デバフを周囲の敵にも伝播させる |
| Righteous Fire | 正義の炎による継続ダメージ |
| Divine Surge | 神聖系ダメージの一斉放出 |

### 装備・武器

- 新防具セット・個別防具ピースをVeilwater Basin各所に追加（セット・単品両方で入手可能）
- **Bloodfeather set**（Patch #11で追加）: Veilwater Basin内の隠し墓に配置。ダガー+レンジャー型ビルド向け
- 全武器のステータス・名称・ビジュアルを見直し。ロアウェポンをいつでもクラフト・現在レベルにスケール可能に
- 武器耐久値がアイテムレベル・レアリティに応じてスケールするように変更
- 新武器: Fell Thunderbrute Greatsword、Scavenger's Acid Cleaver など

### バランス調整

- **リジェネ大幅弱体化**: HP・マナ・スタミナの自然回復量を削減。食料・装備・回復スペルへの依存度が上昇
- **宝箱ファーム制限**: ゴールドチェストの開封状態がセーブされ、即リセットループが廃止
- **武器ダメージパーク調整**: +20%ダメージパークを+10%に引き下げ（Patch #11）。ビルドの多様性を促進
- **スペル**: Fire Boltを追加し序盤の魔法使いを強化

### QoL・システム

- **リスポーン変更**: 死亡時に最寄りの拠点ではなく最後のセーブポイントに復帰するように変更
- **入力カスタマイズ大幅強化**: キーバインディングメニューを刷新。競合検出・メニュー操作対応を追加
- **アイテム詳細UIリワーク**: ステータス・詳細の可読性を大幅向上
- **マップマーカー追加**: 釣りスポット用・水場用の2種類のカスタムマーカーを追加（Patch #11）
- 釣りチュートリアルを全面刷新（Patch #11）
- 水配置器・水ドレインのON/OFFトグル機能（Patch #11）
- NPCおよび動物のパスファインディング改善

### ホットフィックス・追加パッチ

- **Patch #11 v.0.9.0.2（2025-12-01）**: Bloodfeather set追加、Castle家具セット追加、水シミュレーション改善、魔法シールドHP大幅強化（350→1020）など
- **Patch #12 v.0.9.0.3（2025-12-16）**: スタミナ回復値をv0.7以前の水準に戻すリバート、Blood Rose衣装追加（Twitch Drops）、Maritime bathroom家具セット追加、黒曜石ブロック建材コスト削減など

## Update 6: Thralls of Twilight（2025-05-13）

**v.0.8.1.0**

### 新コンテンツ

- **ナイトテンプル（Night Temple）**: 全バイオームに新設された夜限定ダンジョン。昼は扉が封鎖され入れない。トラップと強敵が待ち受け、夜間限定の限定ドロップが報酬として得られる
- **夜咲き植物（Night-Blooming Plants）**: 夜間にのみ各バイオームで出現・採集可能な新植物。新ポーション・食料のクラフト素材になる
- **シュラウド内100カ所超の新探索ポイント**: ランドスケープが刷新され、縦方向の複雑さと探索価値が大幅向上

### 新システム

- **武器ジェム（Weapon Gem）システム**: レア度の高い武器にジェムスロットが付くようになった。強敵のドロップや専用施設「ジェム・フォージ（Gem Forge）」（シュラウド内に点在）でジェムを入手・装着・アップグレード可能
- **アーマー染色（Armor Tinting）**: ほぼ全防具セットが新設の染色ステーションで自由に配色できるようになった。バニティスロットにも対応

### 新NPC

- **エリオ・リッチ床屋（Elio Ricci the Barber）**: 拠点に配置することでキャラクターの外見（髪型・カラー等）をいつでも変更可能。新ヘアスタイルはクエストで追加解放

### 戦闘調整

- ワンドの弾速・精度向上
- 新チャネル型ビームスペル追加
- 一部の敵がカース（呪い）やデバフを付与するようになった（対応する解除アイテムをクラフトで入手可能）
- 夜間の敵パトロール強化によるハイリスク・ハイリターードな夜間探索

### 技術・グラフィック

- シュラウドのボリューメトリックレンダリングを全面刷新（ゴッドレイ表現など視覚品質が大幅向上）
- マルチプレイホスト・専用サーバーのCPUパフォーマンスを大幅最適化
- 照明・シャドウ・大気表現の改善

## Update 5: Pact of the Flame（2025-01-28）

**v.0.8.0.0 / ゲーム1周年記念アップデート**

### 新コンテンツ

- **タウンスフォーク（Townsfolk）**: 拠点に招いて配置できる一般NPCが追加。昼夜サイクルに従って行動し、ベッドで就寝する
- **ダンアルマジロ（Dune Armadillo）**: キンドルウェイスツ（Kindlewastes）に生息する新ペット動物。テイムして農場で繁殖・育成可能
- **70種以上の新アイテム**: 家具・装飾品・建材などを含む大量追加
- **複数の集落リワーク**: 既存集落のレイアウトや内容が刷新

### 建築改善

- **一人称建築カメラ**: 建築ツール使用中に一人称視点に切り替えて精密配置が可能に
- **プロップ全軸回転**: 装飾プロップ（皿・看板等）を全3軸で自由に回転配置可能に（家具は縦軸のみ）
- **オーバーグロースツール**: ツタや草などの装飾植生を建材ブロックに追加・除去できる新建築ツール（ファーマーNPCでクラフト）
- **ブロック劣化ツール**: 壁の端を廃墟風にエイジング加工できる新ツール（コレクターNPCでクラフト）
- **マジックチェストから建築**: バックパックに素材が1個あれば、近隣マジックチェストの素材を直接建築に使用可能

### 社交・コミュニケーション

- **テキストチャット**: マルチプレイヤーでサーバー内テキストチャットが可能に（ホストが有効/無効を切り替え可能）
- **ボイスチャット**: グローバル通話と近接通話を切り替え可能
- **エモートホイール**: Xキー（コントローラーは十字キー下）でアクセス

### ゲームプレイ改善

- **スキル個別リセット**: スキルツリー内で個別スキルのポイント返還が可能に
- **パリィインジケーター追加**: これまでパリィ表示のなかった敵・飛び道具にもパリィタイミング表示が追加
- **農場バランス調整**: 動物の資源生産速度を見直し（スタックサイズを増加して生産間隔を延長）

## Update 2: Melodies of the Mire（v0.7.2.0 / 2024-05-14）

**Early Access第2回大型アップデート。バージョン番号: 532860**

### 新エリア

- **Blackmire（ブラックマイア）**: Revelwoodの北端に広がる新サブバイオーム。対象レベルLv13〜15。高くそびえる木々と暗い湿地が特徴。新クエスト・新敵・新スキルポイント・新音楽・新素材・新装飾アイテムを収録。グラップリングフックとグライダーが前提装備。もう1人の新サバイバーNPCが隠れている

### 新システム

- **プレイヤー個別クエスト進行**: マルチプレイでも個人ごとにクエストを完了できるオプションを追加（最大の要望機能）。クエストジャーナルで未完了クエストの確認・個別攻略が可能
- **楽器（Musical Instruments）**: フルート・リュート等の楽器をクラフト・演奏可能。近くのプレイヤーと合奏もできる。演奏で休息バフ（Rested）の持続時間が延長
- **バニティシステム（Vanity System）**: 装備スロットに外見変更スロットが追加。戦闘ステータスを犠牲にせず見た目を変更可能。グローブ・ヘルメットの非表示オプション追加
- **サーバー権限設定（Permission Settings）**: ユーザーグループごとにパスワードと権限を設定可能。ベース内チェストやブロックへのアクセス、キック・BAN等の管理が可能に

### 新武器タイプ

- **ダガー（Dual-Wielded Daggers）**: 敏捷（Dexterity）スケールの二刀流ダガー。パリィ後のキック攻撃でスタンバーを蓄積。新スキル「Dagger Master」「Slice And Dice」追加

### バランス調整

- **メイジ調整**: Water Auraをバフ型にリワーク（複数スタック不可）。ワンド攻撃のキャンセルタイミングを調整（過剰な防御の最適化を抑制）。新スキル「Eternal Spark」追加（攻撃時に20%の確率で耐久値を消耗しない）
- **ウォリアー強化**: 新スキル「Opportunity」「Finesse」「Steadfast」追加。2H武器の攻撃チェーンを改善（遅延入力で別のオーバーヘッド攻撃チェーンが発動）。Merciless Strikeのベースダメージを+400%→+500%に増加
- **レンジャー拡張**: Magic Arrow（消耗しないがManaを消費）・Explosive Arrow（Revelwoodティアで早期解放）を追加。新スキル「Multi Shot Trigger」「Multi Shot Spread」追加
- クエスト完了でXP獲得できるようになった（それに合わせてレベルアップ必要XPも調整）

### Patch #4（v0.7.2.1 / 2024-06-04）

- アップデート後にベース内ブロックが1ブロックずつしか解体できなくなる問題を修正
- カスタムマップマーカーの位置ずれ問題を修正
- サーバー再参加時のクラッシュ修正
- eyebrow colorがv0.7.3.0で変わっていた問題を修正（全キャラの眉色をニュートラルブラウンにリセット）
