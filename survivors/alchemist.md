# Alchemist（アルケミスト）

> 📌 情報源: [Enshrouded Wiki - Alchemist](https://enshrouded.wiki.gg/wiki/Alchemist) / [Survivors](https://enshrouded.wiki.gg/wiki/Survivors)

## キャラクター概要

- **本名**: Balthazar（バルタザール）
- **役割**: ポーション・魔法系装備・スペル・調合品のクラフト全般
- **救出優先度**: 高（ポーション・魔法ビルドは2番目に救出推奨）

## 救出情報

| 項目 | 内容 |
|------|------|
| 救出場所 | Ancient Vault - Alchemist |
| エリア | Springlands（スプリングランズ）付近 |
| 救出クエスト | Powerful Alchemy |

## 解放されるクラフトステーション

| ステーション | 主な用途 |
|------------|---------|
| 臼（Mortar） | 素材の粉砕・基本調合 |
| 粉砕石（Grinding Stones） | 鉱物の粉末化 |
| 錬金術ステーション（Alchemy Station） | ポーション・スペル・エリクサーの製造 |
| ラボラトリ（Laboratory） | 高品質な調合・上位レシピ |
| 製粉機（Mill / Flour Mill） | 食材・穀物加工 |

## 主要クラフト品

### 錬金術ステーション（Alchemy Station）
| カテゴリ | アイテム例 |
|---------|-----------|
| ポーション | Greater Mana Potion、Greater Mana Potion+、Greater Shroud Survival Flask |
| 素材 | Alchemical Base（錬金術ベース）、Antiseptic（消毒薬） |
| スペル（永久版） | Eternal Fireball / Fireball II、Eternal Ice Bolt / Ice Bolt II、Eternal Heal Beam Channel、Eternal Chain Heal |
| 設備 | Athanor（アタノール）、Improved Millstone（改良製粉石）、Scientific Instruments（科学器具） |

### 魔法装備
魔法系（キャスター）ビルド向けの防具をクラフト可能：
- Alchemist Apprentice（錬金術師見習い）シリーズ各部位
- Archmage（大魔法使い）シリーズ各部位

## Update 8: 装備ルーンアップグレード

Update 8以降、Alchemistは**ワード（Ward）**をルーンでアップグレードできます。

- 対象: ワード（Ward）系装備
- 方法: Alchemistに話しかけ「Upgrade」を選択
- 必要: 対応するルーン素材＋装備品
