# Blacksmith（ブラックスミス）

> 📌 情報源: [Enshrouded Wiki - Blacksmith](https://enshrouded.wiki.gg/wiki/Blacksmith) / [Survivors](https://enshrouded.wiki.gg/wiki/Survivors)

## キャラクター概要

- **本名**: Oswald Anders（オズワルド・アンダース）
- **役割**: 近接系武器・防具・金属部品のクラフト全般
- **救出優先度**: 最優先（武器・防具強化の核心NPC）

## 救出情報

| 項目 | 内容 |
|------|------|
| 救出場所 | Ancient Vault - Blacksmith |
| エリア | Springlands（スプリングランズ） |
| 救出クエスト | Find The Sleeping Survivor |

Springlandsのマップ序盤に位置し、ゲーム開始直後に向かえる最初のVaultの一つ。霧（Shroud）の中にVaultが存在するため、フレイムオルター（Flame Altar）を強化してShroud耐性を上げてから挑むこと。

## 解放されるクラフトステーション

| ステーション | 主な用途 |
|------------|---------|
| 炭焼き窯（Charcoal Kiln） | 木炭の生産（製錬の燃料） |
| 鍛冶炉（Forge） | 武器・防具・金属部品のクラフト |
| 製錬炉（Smelter） | 鉱石→インゴット変換（木炭燃料） |
| 鍛冶道具（Smithing Tools） | 小物・部品クラフト |
| ブラストファーネス（Blast Furnace） | 高品質インゴット精錬 |

## 主要クラフト品

### フォージ（Forge）で作れるもの
- 防具: Adventurer（アドベンチャラー）シリーズ各部位（ヘルメット・チェスト・手甲・ズボン・ブーツ）
- 盾: Hero's Shield（ヒーローの盾）、Ghostly Shield（幽霊の盾）
- 武器: Iron Claw Club（鉄製爪棍棒）、Enhanced Spiked Club（強化棘棍棒）
- 工具: Copper Axe（銅製斧）、Copper Pickaxe（銅製ツルハシ）、Grappling Hook各種
- 設備: Blast Furnace（ブラストファーネス）本体

### 製錬炉（Smelter）で作れるもの
燃料に木炭（Charcoal）が必要。

| インゴット | 素材 |
|-----------|------|
| Copper Bar（銅インゴット） | 銅鉱石 |
| Tin Bar（スズインゴット） | スズ鉱石 |
| Bronze Bar（青銅インゴット） | 銅鉱石＋スズ鉱石 |
| Iron Bar（鉄インゴット） | 鉄鉱石 |
| Glass（ガラス） | 砂 |

## Update 8: 装備ルーンアップグレード

Update 8以降、Blacksmithは**防具・盾**をルーンでアップグレードできます。装備の品質ランク（Common / Uncommon / Rare / Epic / Legendary）がアップグレード可能な段階数を決定します。

- 対象: 防具各部位・各種盾
- 方法: Blacksmithに話しかけ「Upgrade」を選択
- 必要: 対応するルーン素材＋装備品
