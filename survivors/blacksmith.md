# Blacksmith（ブラックスミス）

> 📌 情報源: [Enshrouded Wiki - Survivors](https://enshrouded.wiki.gg/wiki/Survivors)

- **本名**: Oswald Anders（オズワルド・アンダース）
- **救出場所**: Ancient Vault - Blacksmith（主にSpringlandsに存在）
- **役割**: 近接系武器・防具のクラフトとアップグレード

## 解放されるクラフトステーション

| ステーション | 用途 |
|------------|------|
| 炭焼き窯（Charcoal Kiln） | 木炭の生産 |
| 鍛冶炉（Forge） | 武器・防具のクラフト |
| 製錬炉（Smelter） | 鉱石→インゴット変換 |
| 鍛冶道具（Smithing Tools） | 小物・部品クラフト |
| ブラストファーネス（Blast Furnace） | 高品質インゴット精錬 |

## Update 8: 装備アップグレード（ルーン）

Update 8以降、Blacksmithは防具・盾をルーンでアップグレードできます。装備の品質ランクがアップグレード段階数を決定します。
