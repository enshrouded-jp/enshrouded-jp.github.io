# Hunter（ハンター）

> 📌 情報源: [Enshrouded Wiki - Survivors](https://enshrouded.wiki.gg/wiki/Survivors)

- **本名**: Athalan Skree（アサラン・スクリー）
- **救出場所**: Ancient Vault - Hunter
- **役割**: 遠距離系防具の製作・生存装備の提供

## 解放されるクラフトステーション

| ステーション | 用途 |
|------------|------|
| 乾燥ラック（Drying Rack） | 食材・素材の乾燥 |
| 手紡ぎ機（Hand Spindle） | 糸・繊維加工 |
| なめし革ステーション（Tanning Station） | 皮革加工 |
| 紡ぎ車（Spinning Wheel） | 高品質糸の生産 |
| 織機（Loom） | 布・繊維素材の製織 |
