## 装備レアリティ

Update 8（Forging the Path）のパッチノートおよびwikiで判明している内容：

| レアリティ | 主な入手方法 | 特徴 |
|-----------|------------|------|
| Common | ドロップ（基本） | アップグレード回数が最少 |
| Uncommon | ドロップ | — |
| Rare | クラフト（デフォルト）/ ドロップ | クラフト品は基本的にRare品質 |
| Legendary | ドロップ（高難度） | アップグレード回数が最大 |

**重要な仕様:**

- レアリティはベースステータス（ダメージ・耐性値）には**影響しない**
- レアリティはアップグレード（ルーン強化）の**可能回数のみ**に影響する
- ドロップ防具はCommon〜Legendaryの幅があり、クラフト品（Rare）より高品質な防具が入手できる場合がある
- 固有名・特殊効果を持つレジェンダリー武器（Lore武器）はクラフト可能なものも存在する

## Update 8（Forging the Path）の主な変更点

- **武器Special Ability**: 各武器種に固有のUltimate攻撃が追加
- **ルーン強化システム**: 武器だけでなく防具・盾・リングもアップグレード対応
- **Perk追加**: スキルレベル+1系の新Perkが導入
