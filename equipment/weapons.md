# 武器一覧

> 📌 情報源: [enshrouded.wiki.gg/wiki/Weapons](https://enshrouded.wiki.gg/wiki/Weapons) / [hacktheminotaur.com/enshrouded](https://hacktheminotaur.com/enshrouded/) / Update 8 "Forging the Path"（2025年4月21日リリース）

## 武器種の概要

Enshroudedの武器は**近接**・**遠距離**・**魔法**の3系統に分かれる。武器はほとんどがランダムドロップで入手するが、一部クラフト・固定ドロップでも入手できる。

各武器には複数の**Perk（特性）**が付いており、ダメージタイプや特殊効果を決める。

---

## 近接武器（Melee）

### 片手剣（One-handed Sword）

- 軽快な攻撃速度と汎用性が特徴
- 盾と組み合わせてタンク寄りの立ち回りが可能
- 代表武器: Aurora Sword、Dragon Sword、Fang Blade、Eclipsed Blade、Fell Broad Sword

### 両手剣・グレートソード（Two-handed Sword / Greatsword）

- 高ダメージ・広範囲攻撃が強み。攻撃速度はやや遅い
- **Special Ability（Update 8）**: AOE 360度薙ぎ払い。周囲全体を攻撃するUltimate
- 代表武器: Barbarian Greatsword、Chimerical Greatsword、Fool's Demise、Fool's Errand
- 推奨クラス: Warrior / Barbarian

> Spring 2026予定のGreatsword Reworkにより数値・動作が大幅変更される見込み。

### 両手斧（Two-handed Axe）

- 高い物理ダメージと強靭なスタイルが特徴
- 重撃（Heavy Attack）でのよろめかせ性能が高い
- **Special Ability（Update 8）**: フォーカスゲージ満タン時に広域のAOE攻撃を発動。詳細なアニメーション・範囲の公式記述は未確認
- 代表武器: Executioner's Axe、Champion's Axe、Blood Omen Axe、Abyssal Wing Axe、Boreal Axe
- 推奨クラス: Warrior / Barbarian

### 短剣（Dagger）

- 素早い連撃とバックスタブ（Vicious Perk）が強み
- スタミナ消費が少なく手数で攻める武器
- **Special Ability（Update 8）**: 単体への大ダメージバーストアビリティ
- 代表武器: Baroness Daggers、Broodmother's Eye、Crystal Knives、Adorned Drakling Daggers、Drakling Daggers
- 推奨クラス: Assassin / Rogue

### ウォーハンマー・クラブ（War Hammer / Club）

- 鈍器ダメージ特化。敵のスタン・よろめかせが得意
- 重装敵や骨格系エネミーに有効なBlunt（打撃）属性
- **Special Ability（Update 8）**: フォーカスゲージ満タン時に発動する強力な打撃攻撃。詳細な公式記述は未確認
- 代表武器: Battering Ram Hammer、Chieftain's Hammer、Dreadbane、Blessed Club、Frost Smasher、Flail Of Misery
- 推奨クラス: Tank / Warrior

## 遠距離・魔法武器

### 杖（Wand）

- 左手・右手どちらにも装備可能な小型魔法武器
- 連続した魔法弾を発射し、マナ効率が高い
- **Special Ability（Update 8）**: 大型のAOEオーブを放つUltimate
- 代表武器: Apprentice Wand、Blazing Wand、Blizzard Wand、Crackling Wand、Coalescent Wand、Darya's Wand
- 推奨クラス: Wizard / Wand Build

### スタッフ（Staff）

- 両手持ちの大型魔法武器。高威力の魔法を放つ
- 属性攻撃（Fire / Ice / Shock / Shroud）に特化した種類が多い
- **Special Ability（Update 8）**: **Update 8時点では未実装。** パッチノートに「スタッフの変更と改善は将来のアップデートで提供予定」と明記されている
- 代表武器: Blackened Staff、Clearwater Staff、Crystalwater Staff、Elder's Staff、Deceiving Staff
- 推奨クラス: Wizard / Battlemage / Healer

### 弓（Bow）

- 遠距離から高精度のダメージを与える
- 属性矢（Fire/Poison/Ice等）との組み合わせが重要
- **Special Ability（Update 8）**: フォーカスゲージ満タン時に発動する遠距離特殊射撃。詳細な公式記述は未確認
- 代表武器: Antler Bow、Arsonist's Bow、Composite Bow、Dragon's Wing、Forest Longbow、Draconian Bow
- 推奨クラス: Archer / Hunter / Ranger

## Update 8 の武器システム変更点

**Special Ability（武器固有アルティメット）**

- Update 8（Forging the Path、2025年4月21日）で追加された新システム
- 攻撃するごとに専用ゲージが蓄積し、満タン時に発動可能
- ゲージはスキルバー下部中央に表示される
- 通常のHeavy Attackより大幅に強力
- 発動中はスタッガー免疫（Stagger Immunity）を付与
- 武器種ごとに固有のアニメーションと効果を持つ

**Heavy Attack（重攻撃）**

- 攻撃ボタン長押しで発動
- 通常攻撃よりスタミナ消費が多い代わりに高ダメージ
- 小型の敵に対して限定的なスタン効果を付与
- 武器種ごとに異なるアニメーション
