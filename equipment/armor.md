## 防具の基本仕組み

Enshroudedの防具は**Physical Resistance（物理耐性）**と**Magical Resistance（魔法耐性）**を提供する。高レベルになるほど追加効果が解放される。

**装備スロット（5箇所）**

| スロット | 説明 |
|----------|------|
| 頭（Head / Helmet） | 頭部防具 |
| 胴（Upper Body / Chest） | 胴体防具 |
| 手（Arm / Gloves） | 手・腕防具 |
| 脚（Lower Body / Legs） | 脚部防具 |
| 足（Foot / Boots） | ブーツ |

**セットボーナスはない。** 異なるセットを自由に組み合わせることが基本的な立ち回りとなる。

## Update 8 のルーン強化システム（Enhance Equipment）

Update 8（2025年4月21日）で装備強化システムが大幅に改修された。

**対象装備**

- 武器（Weapon）: アップグレード可能
- 鎧（Armor）: Update 8でアップグレード対応追加
- 盾（Shield）: アップグレード対応
- リング（Ring）: アップグレード対応

**仕組み**

- アップグレード回数はアイテムのレアリティによって決まる
- レアリティが高いほどアップグレード可能回数が増える
- アップグレードにより**物理耐性と魔法耐性が同時に上昇**する
- 不要な装備は分解してルーン化できる

**Update 8のその他変更点**

- 基本防御値は低下したが、アップグレードで補填可能
- 新Perkとして「特定スキルレベル+1」ボーナスが追加
- 防具タイプ間の性能差が明確に調整された

## 防具の入手方法

1. **クラフト**: Blacksmith / Hunter / Alchemist NPCに依頼
2. **ドロップ**: 各地域の宝箱・エネミードロップ
3. **クエスト報酬**: 特定クエスト完了時
4. **コスメティック**: Bard NPCまたはクエスト報酬（見た目のみ）
