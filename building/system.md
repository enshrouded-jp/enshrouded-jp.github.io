# 建築システム

> 📌 情報源: [enshrouded.wiki.gg/wiki/Building](https://enshrouded.wiki.gg/wiki/Building) / [enshrouded.wiki.gg/wiki/Flame_Altar](https://enshrouded.wiki.gg/wiki/Flame_Altar) / [enshrouded.wiki.gg/wiki/Construction_Hammer](https://enshrouded.wiki.gg/wiki/Construction_Hammer)

## 建築モードの入り方

1. **コンストラクション・ハンマー（Construction Hammer）** をクラフトする（必要素材: Stone × 1、Manual Crafting で即時製作）
2. ハンマーをホットバーに装備して手に持つ
3. 建築モードに入ると UI が切り替わり、ブロック選択・設置が可能になる

## First Person Mode（一人称視点）

建築モード中に **一人称視点に切り替える**ことで、細かいブロック配置を正確に行える。壁の内側・天井裏など通常視点では配置しにくい箇所に有効。切り替えはデフォルトキー（PC: `T` キー / コントローラー: 情報収集中）。

## ブロック種類

### 建築材料（Building Materials）

| カテゴリ | 素材例 |
|----------|------|
| Wood（木材） | 板材・梁・柱・手すり |
| Stone（石材） | 石ブロック・コブルストーン・磨き石 |
| Metal（金属） | 鉄板・鉄骨 |
| Clay / Brick（レンガ） | 焼きレンガ・タイル |

### 地形素材（Terrain Materials）

地面・自然地形の再配置に使う。掘削した土・岩は素材として回収し、別の場所に再設置できる。

### 屋根素材（Roof Materials）

屋根専用のパーツ。傾斜・平屋根など形状が異なる複数タイプが存在する。

### 植生素材（Overgrowth Materials）

草・苔・葉など装飾的な植生。Overgrowth Tool を使って配置する。

## ブロックサイズ

同一素材でも複数のサイズが存在する：

| サイズ | 用途 |
|--------|------|
| 0.5M | 細かい装飾・補完 |
| 1M | 標準ブロック |
| 2M | 大型壁・床 |
| 4M | 大規模構造物の基礎 |

## 建築ルール

- **Flame Altar 範囲内のみ永続** — 範囲外で設置したブロックは約 30 分またはゲーム再起動でリセットされる
- **範囲の拡大** — Flame Altar をアップグレードすることで建築可能範囲が拡張される（最大 120×120×120 以上）
- **複数 Altar** — Flame Level を上げることで複数の Flame Altar を設置できる。拠点を複数箇所に分散させる際に有効

## voxel 掘削

地形を自由に削ることができ、削った素材は自動的にインベントリへ収納される。掘削した箇所は後から地形素材を再設置して埋め戻すことも可能。

## スナップ・グリッド

- ブロックは基本的にグリッド単位でスナップして配置される
- ハーフスナップ（0.5M 単位）に切り替えることで細かい配置が可能
- 柱・梁・壁などのパーツはスナップ点が設定されており、自動的に整列する

## Magic Chests（マジック・チェスト）— Update 8 追加機能

**Update 8 で追加された新機能。** 建築モード中に付近の **マジック・チェスト（Magic Chest）** を参照できるようになった。

### 使い方

1. 拠点内に Magic Chest を設置・素材を格納しておく
2. 建築モードに入る
3. 付近に Magic Chest がある場合、インベントリの素材欄にチェスト内の素材が表示される
4. インベントリに持ち込んでいなくても、チェスト内の素材を直接建築に使用できる

### メリット

- 大量の建築素材を持ち歩く必要がなくなる
- 大規模建築時の素材管理が大幅に効率化される
- 複数の Magic Chest を連携させることで、巨大な素材プールを建築に活用できる

## Comfort（快適度）

拠点の快適度を示す数値。一定値以上で生存者を拠点に配置できるようになる。

- 家具・装飾品・照明を設置することで Comfort 値が上昇する
- 快適度が高いほど、より多くの生存者が拠点に定着する
- 具体的な必要数値・加算アイテムは情報収集中
