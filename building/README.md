# 建築概要

> 📌 情報源: [enshrouded.wiki.gg/wiki/Building](https://enshrouded.wiki.gg/wiki/Building) / [enshrouded.wiki.gg/wiki/Flame_Altar](https://enshrouded.wiki.gg/wiki/Flame_Altar) / [enshrouded.wiki.gg/wiki/Construction_Hammer](https://enshrouded.wiki.gg/wiki/Construction_Hammer)

Enshrouded は voxel ベースの建築システムを採用しており、地形の掘削・ブロック設置を自由に行える。建築はゲームの核となる要素で、拠点の快適度（Comfort）が生存者の呼び寄せにも影響する。

## 建築の基本

- **コンストラクション・ハンマー（Construction Hammer）** を装備して建築モードに入る
- **Flame Altar（フレイム・アルター）** の影響範囲内のみ、建築・地形変更が永続化される
- 範囲外での変更は約 30 分またはゲーム再起動でリセットされる
- 建築モード中は **一人称視点（First Person Mode）** への切り替えが可能

## ツール一覧

| ツール | 用途 |
|--------|------|
| Construction Hammer | ブロック設置・建築の基本ツール |
| Overgrowth Tool | 植生（草・葉など）の配置 |
| Decay Tool | ブロックを朽ちた状態に変化させる装飾用ツール |

## Flame Altar の建築範囲

| レベル | 範囲（最大サイズ） | アップグレード素材 |
|--------|----------------|----------------|
| Lv.1 | 40×40×40 | — |
| Lv.2 | 80×80×80 | Shroud Core × 1 |
| Lv.3 | 120×120×120 | Shroud Core × 5 |
| Lv.4 以降 | さらに拡大 | Shroud Nexus 等 |
